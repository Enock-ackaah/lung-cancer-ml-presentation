# Predicting Lung Cancer Using Machine Learning

This repository contains a class presentation on predicting lung cancer using multiple machine learning models applied to a publicly available Kaggle dataset.

## Methods Used
- Logistic Regression
- Decision Trees (Pruning, Bagging, Boosting)
- Random Forest
- Support Vector Machines (Linear, Polynomial, RBF)
- Neural Network

## Evaluation Metrics
- AUC (ROC)
- RMSE
- Confusion Matrix

## Key Findings
- Linear SVM and Neural Network models achieved strong test performance
- Linear SVM provided the best balance between accuracy and interpretability
